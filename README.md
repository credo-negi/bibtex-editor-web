# BibTeX Editor Web

このプロジェクトは、BibTeXファイルを簡単に編集・管理するためのWebアプリケーションです。

## 特徴

- BibTeXファイルのインポートとエクスポート
- エントリの追加、編集、削除、検証

## インストール

以下のコマンドを実行して依存関係をインストールします。

```bash
npm install
```

## 使用方法

開発サーバーを起動するには、以下のコマンドを実行します。

```bash
npm run dev
```

ブラウザで `http://localhost:5173` にアクセスしてアプリケーションを使用できます。
